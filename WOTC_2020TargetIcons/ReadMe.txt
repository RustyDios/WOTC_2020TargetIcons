[h1]What is this?[/h1]
An entire suite of new target icons, mostly for modded enemy units.

[h1]So what is included?[/h1]
The mod currently provides [b](86)[/b] new icons for the following mods;
[list]
[*]CX enemy mods
[*]ABA Primes
[*]Faction Heroes
[*]Claus' Enemies
[*]Elite Aliens
[*]Muton Centurions
[*]A Harder War
[*] ... and many more
[/list]

[h1]Compatibility/Known Issues[/h1]

[b]Works perfectly fine with dotvhs' [WOTC] New Target Icons mod[/b]
From user feedback; works with LWotC
No known issues, bug reports welcome

[h1]Credits and Thanks[/h1]
I re-use and adapted some icons from dotvhs' WOTC work, permission was obtained before use.

Most new icons created by NelVlesis and code work done by myself

I'd like to give thanks to NelVlesis for doing all the heavy work :)
Tips, tweaks and inspirations from MrShadowCX and also for putting this small team together to get these made.
Special thanks also go to CX, Iridar and RoboJumper for helping with coded compatibility for dotvhs' icon pack, which I [b]heartily recommend as a companion mod;[/b][list]
[*][url=https://steamcommunity.com/sharedfiles/filedetails/?id=1384631824] WOTC New Target Icons[/url]
[/list]
A huge shoutout to dotvhs for giving permission to adapt their icons and for the original inspiration in replacing target icons!
